# idcbr

## Installation
* Create repository on Github
* Install Sergey
* Install Bootstrap


**Terminal commands:**
```
git clone https://github.com/kkmachado/[repository].git
npm init -y
npm install sergey
npm install bootstrap
```


**Edit package.json**
```
"scripts": {
   "start": "sergey",
   "dev": "sergey --watch"
}
```


## Start server
`npm run dev`


## Build
`npm start`


## Deploy
Commit to Github and connect repository with netlify.com
